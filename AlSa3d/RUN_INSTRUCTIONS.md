# 🚀 دليل تشغيل برنامج Al-Sa3d للمحاسبة

## المتطلبات الأساسية

### 1. تثبيت SQL Server
- **SQL Server 2019** أو أحدث (Express أو Developer مجاني)
- أو **SQL Server LocalDB** (مدمج مع Visual Studio)

### 2. تثبيت .NET SDK
- **.NET 8.0 SDK** أو أحدث
- تحميل من: https://dotnet.microsoft.com/download

### 3. Visual Studio 2022 (اختياري لكن موصى به)
- مع Workload: **.NET Desktop Development**

---

## خطوات التشغيل

### الخطوة 1️⃣: إنشاء قاعدة البيانات

```bash
# افتح SQL Server Management Studio أو Azure Data Studio

# 1. أنشئ قاعدة البيانات
sqlcmd -S localhost -E -i "database/01_Create_Database.sql"

# 2. أنشئ الجداول
sqlcmd -S localhost -E -i "database/02_Create_Tables.sql"

# 3. أضف البيانات التجريبية
sqlcmd -S localhost -E -i "database/03_Seed_Data.sql"

# 4. أضف الإجراءات المخزنة
sqlcmd -S localhost -E -i "database/04_Stored_Procedures.sql"

# 5. أضف الدوال والعروض
sqlcmd -S localhost -E -i "database/05_Views_Functions.sql"
```

**أو من SQL Server Management Studio:**
1. افتح كل ملف `.sql` بالترتيب
2. اضغط `F5` للتنفيذ
3. انتظر رسالة النجاح قبل الانتقال للتالي

---

### الخطوة 2️⃣: تحديث Connection String (إذا لزم الأمر)

افتح ملف `appsettings.json` في مجلد `AlSa3d.Desktop`:

```json
{
  "ConnectionStrings": {
    "DefaultConnection": "Server=localhost;Database=AlSa3d;Trusted_Connection=True;"
  }
}
```

**غيّر `localhost` إلى اسم السيرفر الخاص بك إذا كان مختلفاً.**

---

### الخطوة 3️⃣: بناء وتشغيل التطبيق

#### الطريقة أ: من Visual Studio
```
1. افتح AlSa3d.sln في Visual Studio
2. اضغط F6 للبناء (Build)
3. اضغط F5 للتشغيل (Run)
```

#### الطريقة ب: من Command Line
```bash
cd src/AlSa3d.Desktop

# بناء المشروع
dotnet build

# تشغيل التطبيق
dotnet run
```

#### الطريقة ج: نشر وتشغيل كملف مستقل
```bash
# نشر للتطبيق
dotnet publish -c Release -o ./publish

# تشغيل الملف المنشور
./publish/AlSa3d.Desktop.exe
```

## استكشاف الأخطاء

### ❌ خطأ في الاتصال بقاعدة البيانات
```
الحل:
1. تأكد من تشغيل SQL Server
2. تحقق من Connection String في appsettings.json
3. جرب استخدام Windows Authentication
4. تأكد من وجود قاعدة البيانات AlSa3d
```

### ❌ خطأ في بناء المشروع
```
الحل:
1. تأكد من تثبيت .NET 8 SDK
2. نفذ: dotnet restore
3. نفذ: dotnet clean
4. نفذ: dotnet build --no-incremental
```

### ❌ الشاشات لا تظهر
```
الحل:
1. تحقق من وجود جميع ملفات XAML
2. تأكد من بناء المشروع بنجاح
3. امسح مجلد bin و obj وأعد البناء
```

### ❌ البيانات لا تظهر
```
الحل:
1. تأكد من تنفيذ Seed_Data.sql
2. تحقق من وجود بيانات في الجداول
3. راجع Logs/app-.log للأخطاء
```
