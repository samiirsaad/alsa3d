# 🏗️ Al-Sa3d - أكبر برنامج محاسبة في مصر

## 📋 نظرة عامة على المشروع

**Al-Sa3d** هو نظام محاسبي متكامل وحديث مبني بأحدث التقنيات (2026)، مصمم ليكون أكبر وأقوى برنامج محاسبة في مصر.

### 🎯 الرؤية
بناء نظام محاسبي متكامل يجمع بين:
- ✅ قوة وثبات أنظمة المحاسبة التقليدية
- ✅ حداثة ومرونة التقنيات المعاصرة
- ✅ واجهات مستخدم سهلة واحترافية
- ✅ أمان عالي للبيانات المالية
- ✅ قابلية التوسع والنمو

---

## 🚀 التقنيات المستخدمة

### الـ Backend
- **اللغة**: C# 12 (.NET 8)
- **قاعدة البيانات**: SQL Server 2022 / PostgreSQL 16
- **ORM**: Entity Framework Core 8
- **API**: ASP.NET Core Web API

### الـ Frontend (Desktop)
- **الإطار**: WPF (Windows Presentation Foundation)
- **UI Library**: Material Design in XAML
- **Reporting**: DevExpress / Crystal Reports
- **Barcode**: ZBar / ZXing.Net

### البنية المعمارية
- **Pattern**: MVVM (Model-View-ViewModel)
- **Architecture**: Clean Architecture
- **DI**: Microsoft Dependency Injection
- **Logging**: Serilog
